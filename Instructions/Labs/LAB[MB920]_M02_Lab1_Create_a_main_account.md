# Схема обучения 2. Изучение основ Microsoft Dynamics 365 Finance
# Модуль 2. Описание общих реестров

## Лаборатория 2.1. Создание основной учетной записи

## Исходные условия выполнения лабораторной работы

   - **Предполагаемое время**: 5 минут

## Цель

В этой лаборатории вы выполните следующие действия:

1. Создайте юридическое лицо.

2. Создайте финансовый календарь.

3. Создайте диаграмму учетных записей.

4. Создайте основные учетные записи в диаграмме учетных записей.

5. Создайте операционную единицу отдела типов и центра затрат.

6. Создайте структуру учета с помощью основных учетных записей и центров затрат.

7. Настройте реестр.

# Этапы лабораторной работы

## Создание юридического лица

1. В левой области навигации Dynamics 365 Finance выберите **"Организации администрирования ****&gt; &gt; модулей**&gt;** **** организации юридические лица".**

2. **На странице "Юридические лица"** нажмите **кнопку "Создать**" в области действий, чтобы создать новую юридическую сущность, и введите следующие сведения:

    - Имя: **демонстрационные системы Contoso**

    - Компания: **CDS**

    - Страна или регион: **США**

3. Нажмите кнопку **"ОК** " внизу.

## Создание финансового календаря

1. В левой области навигации Dynamics 365 Finance выберите **"Календари общих реестров**&gt;** **** модулей" (Календари общей книги **&gt;** &gt;) "Финансовые календари".** 

2. **На странице "Финансовый календарь**" нажмите **кнопку "Создать календарь**" в области действий, чтобы создать новый финансовый календарь, и введите следующие сведения:

    - Календарь: **CDS**

    - Описание: **календарь демонстрационных систем**

    - Начало финансового года: **10.1.2023**

    - Конец финансового года: **9.30.2023**

    - Название финансового года: **2023-24**

    - Длина периода: **1**

    - Единица: **месяцы**

3. Выберите кнопку **Создать**.

Система создаст 14 периодов, включая один открытый и один закрывающий период и 12 периодов в течение 12 месяцев.

## Создание диаграммы учетных записей

1. В левой области навигации Dynamics 365 Finance выберите **диаграмму общего реестра модулей > ********** **&gt;для диаграммы учетных **&gt;** &gt; записей учетных записей.**

2. **На странице "Диаграмма учетных записей**" нажмите **кнопку "Создать**" в области действий, чтобы создать новую диаграмму учетных записей, и введите следующие сведения:

    - **Диаграммы учетных записей**: демонстрационные системы

    - **Описание**: демонстрационные системы Contoso

3. **Нажмите кнопку "Создать"** на главной **учетной записи FastTab, чтобы создать основные учетные** записи со следующими значениями:

    - Основная учетная запись: **1000**

    - Имя: **наличные деньги**

    - Основной тип расходов: **Балансовый лист**

    - Категория основной учетной записи: **CASH**

    - Дебет/кредит по умолчанию: **Дебет**

4. Создайте другую основную учетную запись:

    - Основная учетная запись: **3000**

    - Имя: **доход**

    - Основной тип расходов: **Доход**

    - Категория основной учетной записи: **REV**

    - Db/CR по умолчанию: **кредит**

5. Создайте другую основную учетную запись:

    - Основная учетная запись: **6000**

    - Имя: **расходы на поездку**

    - Основной тип расходов: **Expense**

    - Категория основной учетной записи: **EXP**

    - Дебет/кредит по умолчанию: **Дебет**

##  Создание операционной единицы

1. Перейдите к **подразделениям администрирования **&gt;** организации modules.**&gt;** **&gt;** **

2. В области действий нажмите **кнопку "Создать** ", а затем **тип операционной** единицы Отдела и введите следующее значение:

    - Имя: **CDS_Training**

3. В области действий нажмите кнопку "Создать", а затем тип операционной единицы центра затрат и введите следующее значение:

    - Имя: **CDS_Purchase**

4. Добавьте еще два центра затрат: **CDS_IT** и **CDS_Admin**.

## Создание структуры учета

1. **На домашней странице "Финансы и операции**" в правом верхнем углу убедитесь, что вы работаете с компанией **CDS**.

2. При необходимости выберите компанию и в меню выберите **CDS**.

3. В левой области навигации Dynamics 365 Finance выберите **"Общие диаграммы**** > реестра ****** **&gt;модулей" структур учетных записей &gt; &gt; настройка структур учетных записей.**

4. В области действий нажмите **кнопку "Создать** ", чтобы создать новую структуру учетной записи со следующими значениями:

    - Структура учетной записи: **CDS_BS**

    - Описание: **баланс CDS**

    - Добавление основной учетной записи: **ДА**

5. Выберите кнопку **Создать**.

6. **В сегментах и разрешенных значениях** FastTab нажмите **кнопку "Добавить**", чтобы добавить **диапазон основной учетной записи 1000..2999**.

7. Нажмите кнопку "Активировать **" **в области действий, **а затем кнопку "Активировать**" в диалоговом окне пакетной обработки.

8. В области действий страницы структур** учетной **записи нажмите **кнопку "Создать**", чтобы создать другую структуру учетной записи со следующими значениями:

    - Структура учетной записи: **CDS_Revenue**

    - Описание: **доход CDS**

    - Добавление основной учетной записи: **ДА**

9. Выберите кнопку **Создать**.

10. **В сегментах и разрешенных значениях** FastTab нажмите **кнопку "Добавить**", чтобы добавить **диапазон основной учетной записи 3000..5999**.

11. Нажмите кнопку "Активировать **" **в области действий, **а затем кнопку "Активировать**" в диалоговом окне пакетной обработки.

12. В области действий страницы структур** учетной **записи нажмите **кнопку "Создать**", чтобы создать другую структуру учетной записи со следующими значениями:

    - Структура учетной записи: **CDS_Expense**

    - Описание: **расходы CDS**

    - Добавление основной учетной записи: **ДА**

13. Выберите кнопку **Создать**.

14. **В сегментах и разрешенных значениях** FastTab нажмите **кнопку "Добавить**", чтобы добавить **диапазон основной учетной записи 6000..9999**.

15. Нажмите кнопку **"Сегмент"** в **сегментах и допустимых значениях** FastTab

16. **В диалоговом окне "Добавить сегмент**" выберите **CostCenter**, а затем кнопку **"Добавить сегмент**".

17. **В поле CostCenter** введите **253.255**. 

18. Нажмите кнопку "Активировать **" **в области действий, **а затем кнопку "Активировать**" в диалоговом окне пакетной обработки.

## Настройка книги учета

1. **На домашней странице "Финансы и операции**" в правом верхнем углу убедитесь, что вы работаете с компанией **CDS**.

2. При необходимости выберите компанию и в меню выберите **CDS**.

3. В левой области навигации Dynamics 365 Finance выберите **страницу** реестра "Общие реестры**** > модулей &gt; &gt;" и настройте следующее:

    - Диаграмма учетных записей: **демонстрационные системы**

    - Финансовый календарь: **CDS**

    - Валюта учета: **USD**

    - Валюта отчетов: **USD**

    - Тип валютного курса учета: **по умолчанию**

    - Тип валютного курса бюджета: **бюджет**

4. **В структуре** учетной записи FastTab нажмите **кнопку "Добавить**", чтобы добавить структуру **учетной записи CDS_BS**.

5. Добавьте еще две структуры учетных записей: **CDS_Revenue** и **CDS_Expense**.

Это завершает базовую конфигурацию модуля общего реестра. Теперь вы можете запланировать конфигурацию журнала для публикации журналов.
